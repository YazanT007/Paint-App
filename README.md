# Paint App 🎨

## Description
A polished paint app built with React and Vite that lets you sketch, create shapes, and place editable text boxes on the canvas.

## Features
- **Pencil, Eraser, and Clear** – Draw freely, erase parts of the canvas, or clear everything when you want to start over.  
- **10 Default Colors** – Quickly choose from 10 predefined colors.  
- **Color Picker** – Select any color you like using the color picker.  
- **Line Width Meter** – Adjust the thickness of your brush or eraser.  
- **Shape Tools** – Draw clean lines, rectangles, and circles.  
- **Undo / Redo** – Step backward or forward through your recent edits.  
- **Multi-Box Text Tool** – Add multiple text boxes, move them around the canvas, resize them, and delete them individually.  
- **Save / Export** – Download your drawing as a PNG image.

## Tech Used
- React.js with Vite  
- CSS  
- react-icons
