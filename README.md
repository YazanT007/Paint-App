# Paint App 🎨

## Description
A simple Paint App that allows you to draw freely on the canvas. Perfect for quick sketches or doodles.

## Features
- **Pencil, Eraser, and Delete Button** – Draw, erase, or clear the entire canvas.  
- **10 Default Colors** – Quickly choose from 10 predefined colors.  
- **Color Picker** – Select any color you like using the color picker.  
- **Line Width Meter** – Adjust the thickness of your brush or eraser.

## Tech Used
- React.js (with Vite)  
- CSS
